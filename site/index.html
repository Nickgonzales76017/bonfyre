Architecture

Four layers. Every binary declares its behavioral class.

Surface cms · api · auth · pipeline · cli · transcript-family · project
Value offer · gate · meter · ledger · finance · outreach · pay · pack · distribute
Transform media-prep · transcribe · transcript-clean · paragraph · brief · proof · embed · narrate · render · emit · mfa-dict · weaviate-index
Substrate ingest · hash · index · compress · stitch · graph · runtime · queue · sync
Libraries libbonfyre (runtime contract, operators, SHA-256) · liblambda-tensors (family compression)

Transform binaries are pure — same inputs, same outputs, cacheable. Surface binaries own state and serve HTTP. Full architecture doc →

Install

Build from source in under 60 seconds.

# From source (recommended)
git clone https://github.com/Nickgonzales76017/bonfyre.git
cd bonfyre
make # builds 2 libraries + 37 binaries
make install # copies to ~/.local/bin
# One command (macOS / Linux)
curl -fsSL https://raw.githubusercontent.com/Nickgonzales76017/bonfyre/main/install.sh | sh

Requirements: C11 compiler (gcc or clang), SQLite3 dev headers, zlib.
